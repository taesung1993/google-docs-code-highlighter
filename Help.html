도움말

코드 블럭을 VSCode 스타일 다크박스(줄번호 + 구문강조)로 바꿔줍니다.

  1. 문서에서 코드 부분을 드래그로 선택합니다.
  2. 메뉴 🎨 Code Highlighter → Apply를 누릅니다.
  3. 팝업에서 언어를 입력/검색하고 (예: Java, Flutter, React), Apply를 누릅니다. (비워두면 자동 감지)
  • 이미 변환한 박스 + 새 줄을 함께 선택해 다시 적용하면 하나로 합쳐집니다.
  • 둥근 따옴표(“ ”)는 곧은 따옴표로 자동 교정돼요.
  • 긴 줄은 자동으로 줄바꿈되며 각 줄에 번호가 붙습니다.

Turns code into a VSCode‑style dark box with line numbers and syntax highlighting.

  1. Select your code in the document by dragging.
  2. Open 🎨 Code Highlighter → Apply from the menu.
  3. In the popup, type or search a language (e.g. Java, Flutter, React) and click Apply. (Leave it blank for auto‑detect.)
  • Select an existing box + new lines and apply again to merge them into one.
  • Curly quotes (“ ”) are auto‑corrected to straight quotes.
  • Long lines wrap automatically, and every line gets its own number.
