Privacy Policy — Code Highlighter for Docs

Last updated: June 16, 2026

Code Highlighter for Docs ("the add‑on") is a Google Docs editor add‑on that converts selected code in your document into a styled, syntax‑highlighted box. This policy explains what the add‑on accesses and how it handles your data.

1. What the add‑on accesses

The add‑on uses the https://www.googleapis.com/auth/documents.currentonly scope, which grants access only to the Google Document that is currently open while you are using the add‑on. It reads the text you select and writes the formatted result back into that same document. It cannot access any of your other documents, your Drive, your email, or your account information.

2. How your data is used

When you run the add‑on, it reads the code you selected, applies syntax highlighting, and replaces the selection with a formatted table — entirely inside Google's Apps Script environment. All processing (including the bundled highlight.js library) runs within Google's servers as part of the document edit.

3. Data storage and sharing

The add‑on does not collect, store, log, sell, or transmit your document content or any personal information to the developer or to any external server or third party. No analytics or tracking is used. Your content never leaves your Google account.

4. Third‑party services

The add‑on does not send your data to any third‑party service. The highlight.js library is bundled and executed locally within the script; no network request is made with your content.

5. Children's privacy

The add‑on is a general productivity tool and is not directed to children under 13.

6. Changes to this policy

If this policy changes, the updated version will be posted at this URL with a new "Last updated" date.

개인정보처리방침 — Code Highlighter for Docs

최종 수정일: 2026년 6월 16일

Code Highlighter for Docs(이하 "애드온")는 Google Docs에서 선택한 코드를 구문 강조된 박스로 변환하는 에디터 애드온입니다.

1. 접근 범위

애드온은 documents.currentonly 권한만 사용하며, 이는 사용 중인 현재 문서에만 접근합니다. 선택한 텍스트를 읽고 변환 결과를 같은 문서에 씁니다. 다른 문서·Drive·이메일·계정 정보에는 접근하지 않습니다.

2. 데이터 사용

선택한 코드를 읽어 구문 강조를 적용하고 표로 교체하는 모든 처리는 Google Apps Script 환경 안에서 이루어집니다. 내장된 highlight.js도 스크립트 내부에서 실행됩니다.

3. 저장 및 공유

애드온은 문서 내용이나 개인정보를 수집·저장·기록·판매·외부 전송하지 않습니다. 분석·추적도 사용하지 않으며, 콘텐츠는 사용자의 Google 계정을 벗어나지 않습니다.

4. 제3자 서비스

사용자 데이터를 제3자 서비스로 보내지 않습니다. highlight.js는 스크립트 내부에서 로컬로 실행되며, 콘텐츠를 담은 네트워크 요청을 하지 않습니다.
