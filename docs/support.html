How to use the add-on

  1. Open a Google Doc and select the code you want to format.
  2. Open the 🎨 Code Highlighter menu and choose Apply.
  3. Pick a language. Your selection becomes a VSCode-style dark code box with line numbers and syntax highlighting.

An in-app Help guide (Korean / English) is available from the same menu.
